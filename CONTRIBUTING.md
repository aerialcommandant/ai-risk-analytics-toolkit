# Contributing

Thank you for your interest in contributing to AI Risk Analytics Toolkit.

## How to Contribute

You can contribute by:

- Improving documentation
- Adding sample datasets
- Writing tests
- Improving risk scoring logic
- Reporting bugs
- Suggesting new analytics workflows

## Pull Request Guidelines

Before submitting a pull request:

1. Explain the problem being solved.
2. Keep changes focused.
3. Avoid committing secrets or sensitive data.
4. Add comments where the logic may not be obvious.
5. Update documentation where necessary.
